# # 🇳🇬 Nigerian Mutual Fund ROI Tracker

##  Project Overview
As an independent researcher and data analyst based in Ibadan, I developed this tool to bridge the gap between structural economic analysis and practical personal finance. This project automates the retrieval and cleaning of raw Net Asset Value (NAV) data from the **Securities and Exchange Commission (SEC) Nigeria**.

It is specifically designed for retail investors in Nigeria who wish to track "real" ROI and manage monthly investments ranging from **₦5,000 to ₦50,000**.

##  Technical Stack
- **Language:** Python 3.x
- **Libraries:** Pandas, BeautifulSoup4, Requests, Openpyxl
- **Deployment:** Streamlit Cloud
- **Data Source:** [SEC Nigeria Monthly NAV Reports](https://sec.gov.ng/)

##  Key Features
- **Automated Scraping:** Automatically fetches the latest 2026 monthly Excel reports.
- **Retail Filter:** Separates institutional funds from retail-friendly funds (unit prices < ₦10,000).
- **Investment Simulator:** Calculates unit purchases based on a sliding monthly budget.

##  Economic Context
This project serves as a practical extension of my research into the **Debt-Growth Nexus** in Nigeria. By analyzing Mutual Fund performance, we observe how private capital allocation responds to the current inflationary environment.
